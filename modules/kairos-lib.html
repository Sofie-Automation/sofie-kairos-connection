    Module kairos-lib

    KAIROS-connection

    Note: This library is not affiliated with Panasonic in any way.

    A small utilities and types library for the kairos-connection package. It exposes shared TypeScript types and helper utilities so projects that cannot or should not pull in the full connection package can still interact with Kairos types and references.

    More documentation can be found here: github.com/Sofie-Automation/sofie-kairos-connection
